# 第一人称武器展示与使用设计草案

## 项目目标

做一个可扩展的第一人称武器展示项目，支持不同武器的展示、检视和基础使用行为。

## 初始体验

- 玩家以第一人称进入展示场景。
- 可以查看当前武器模型。
- 后续可以切换不同武器。
- 每把武器可以有独立的展示信息、动作和音效。

## 后续功能候选

- 第一人称控制器和相机晃动。
- 武器持握、检视、开火、换弹动画。
- 展示柜或靶场场景。
- 武器属性面板。
- Blender 到 Godot 的自动导出和同步工具。

## 当前非目标

- 不追求真实武器模拟。
- 不实现多人联机。
- 不实现完整战斗 AI。
- 不在初始阶段制作大量真实模型。
